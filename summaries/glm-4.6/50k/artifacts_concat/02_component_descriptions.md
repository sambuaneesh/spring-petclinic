
| Component Name | Responsibility | Interfaces (key endpoints or methods) | Depends On (other modules) | Technologies (frameworks, DBs, patterns) |
|---|---|---|---|---|
| Owner Service | Manages owner data, including CRUD operations, search by last name with pagination, and linking to pets. | `GET/POST /owners` | Data Access Layer | Spring Boot, Spring MVC, JPA, Repository Pattern, Form Validation (Jakarta Validation) |
| Pet Service | Handles pet management (CRUD) within the context of an owner. Enforces validation rules like unique pet names per owner and valid birth dates. | `GET/POST /owners/{ownerId}/pets` | Data Access Layer, Catalog Service | Spring Boot, Spring MVC, JPA, Custom Validators (`PetValidator`), Form Validation |
| Vet Service | Provides listings of veterinarians and their specialties. Supports both HTML and JSON responses and implements caching for performance. | `GET /vets.html`, `GET /vets` | Data Access Layer, Catalog Service | Spring Boot, Spring MVC, JPA, Caching (Caffeine, JCache API), `@Cacheable` |
| Visit Service | Manages the scheduling and recording of pet visits. Validates that visit dates are not in the future. | `GET/POST /owners/{ownerId}/pets/{petId}/visits` | Data Access Layer, Owner Service, Pet Service | Spring Boot, Spring MVC, JPA, Form Validation |
| Catalog Service | Serves as a read-only repository for reference data like pet types and vet specialties. | `PetTypeRepository`, `SpecialtyRepository` (internal interfaces) | Data Access Layer | Spring Data JPA, JPA |
| Web UI (Frontend) | Renders the user interface using Thymeleaf templates, styled with Bootstrap/SCSS. Handles internationalization and global error display. | Thymeleaf Templates (`/templates/*.html`) | Owner Service, Pet Service, Vet Service, Visit Service, Catalog Service | Thymeleaf, Bootstrap CSS, SCSS (`libsass-maven-plugin`), Responsive Design, Internationalization (i18n) |
| Data Access Layer | Abstracts all database interactions using Spring Data JPA. Supports multiple databases (H2, MySQL, PostgreSQL) via profiles and manages transactions. | `OwnerRepository`, `PetRepository`, `VetRepository`, `VisitRepository`, `PetTypeRepository`, `SpecialtyRepository` | Database (H2/MySQL/PostgreSQL) | Spring Data JPA, JPA (Entity-Relationship Mapping), Hibernate (implied), Multi-DB Support, `@Transactional`, Pagination |
